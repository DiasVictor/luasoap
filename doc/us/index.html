Overview

LuaSOAP is a library to ease the use of SOAP. LuaSOAP provides a very simple API that convert Lua tables to and from XML documents.

LuaSOAP is free software and uses the same license as Lua 5.0.

Status

Current version is 1.0 beta. It was developed for Lua 5.0 based on LuaExpat.

Download

LuaSOAP can be downloaded in source code from its LuaForge page.

History

  • [2/Dec/2004] Version 1.0 beta released
  • [4/Apr/2004] Version 1.0 alpha released

LuaSOAP Version 1.0 beta follows the package model for Lua 5.1 (see section Installation for more details).
