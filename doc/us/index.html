Overview

LuaSOAP is a library to ease the use of SOAP. LuaSOAP provides a very simple API that convert Lua tables to and from XML documents.

LuaSOAP is free software and uses the same license as Lua 5.0.

Current version

Current version is 1.0 alpha.

Download

LuaSOAP can be downloaded in source code from the following links:

luasoap-1.0a.tar.gz

What's new

  • [] Version 1.0 alpha released

Installation

LuaSOAP is a Lua library so the installation is very simple: just copy the Lua files to a directory in your LUA_PATH.

History

LuaSOAP was designed by Roberto Ierusalimschy, André Carregal and Tomás Guisasola for The Kepler Project that keeps its copyright. It was implemented by Tomás Guisasola.

LuaSOAP development was sponsored by Fábrica Digital and FINEP.
